# mesto-ad
https://github.com/Mertozqq/mesto-production

## Команды

- `npm run dev` — локальный запуск проекта
- `npm run build` — сборка проекта
- `npm run deploy` — деплой собранного проекта

## Деплой

Автоматический: при пуше в `main` через GitHub Actions в репозиторий [mesto-production](https://github.com/Mertozqq/mesto-production).

Ручной: `npm run deploy`.
